## 功能介绍

### 1. Dashboard (仪表盘)
首页概览，展示：
- 产品总数、报告数量、平台数量
- Google Trends 趋势图
- 最近报告列表
- 最近添加的产品

**亮点功能**: **一键选爆品** 按钮，点击后自动分析并显示 Top 10 热门产品。

### 2. Products (产品搜索)
搜索和浏览各平台产品数据。

### 3. Trends (趋势分析)
使用 Google Trends 分析关键词热度趋势。

### 4. Reports (报告中心)
生成和管理选品分析报告。

## 操作指南

### 一、产品搜索

#### 搜索已有产品
1. 进入 **Products** 页面
2. 在搜索框输入关键词（如 "wireless earbuds"）
3. 可选择平台筛选（eBay AU、eBay NZ、Amazon AU、TradeMe）
4. 可选择排序方式
5. 点击 **Search Database** 搜索

#### 从平台抓取新产品
1. 输入搜索关键词
2. 点击 **Fetch eBay AU** 或 **Fetch eBay NZ**
3. 系统会从平台抓取数据并保存到数据库
4. 抓取完成后自动显示结果

#### 查看产品详情
- 点击产品标题查看详情弹窗
- 点击链接图标直接跳转到平台购买页面

#### 生成产品报告
- 点击产品行的 **报告图标** 生成该产品的选品报告

---

### 二、一键选爆品

这是最核心的功能，帮助快速发现热门产品。

#### 使用方法
1. 进入 **Dashboard** 页面
2. 点击红色的 **一键选爆品** 按钮
3. 系统自动分析所有产品，计算热度评分
4. 显示 Top 10 热门产品列表

#### 热度评分算法
```
热度评分 = 评论数(40%) + 评分(30%) + 价格合理性(30%)
```

- **评论数**: 评论越多，说明销量越好
- **评分**: 评分越高，产品质量越好
- **价格合理性**: 10-200 价格区间的产品更受青睐

#### 查看产品
- 点击产品名称直接跳转到平台购买页面
- 查看热度评分判断产品潜力

---

### 三、Google Trends 分析

#### 分析单个关键词
1. 进入 **Trends** 页面
2. 输入关键词（如 "bluetooth speaker"）
3. 选择地区（澳大利亚 / 新西兰）
4. 点击 **Analyze** 分析

#### 查看结果
- **趋势图**: 显示过去 12 个月的搜索热度变化
- **相关搜索**: 显示热门和上升的相关搜索词
- 利用相关搜索词发现更多选品机会

#### 比较多个关键词
1. 输入多个关键词，用逗号分隔（如 "earbuds,headphones,speaker"）
2. 点击 **Compare Keywords**
3. 查看关键词热度对比图

---

### 四、报告生成

#### 创建新报告
1. 进入 **Reports** 页面
2. 点击 **生成新报告** 按钮
3. 填写报告信息：
   - **标题**: 报告名称
   - **类型**: 快速报告 / 完整报告 / 对比报告
   - **目标类型**: 关键词 / 分类 / 产品
   - **目标值**: 具体的关键词或分类名
4. 点击 **Generate** 生成

#### 报告类型说明
| 类型 | 内容 | 适用场景 |
|------|------|----------|
| Quick | 基础数据 + 简要分析 | 快速决策 |
| Full | 完整数据 + 详细分析 | 深度研究 |
| Comparison | 多产品对比分析 | 选择最佳产品 |

#### 查看报告
- 点击眼睛图标查看报告详情
- 查看综合评分、分析结论、建议
- 可下载 PDF 或分享给团队

## 常见问题

### Q: 为什么搜索结果为空？
A: 数据库中可能没有相关产品。请使用 **Fetch eBay AU/NZ** 按钮从平台抓取数据。

### Q: Trends 分析失败怎么办？
A: Google Trends API 有请求限制。等待几分钟后重试，或换个关键词。

### Q: 热度评分怎么解读？
A: 评分范围 0-100，一般来说：
- 70+ 分: 优秀，值得关注
- 50-70 分: 良好，可以考虑
- 50 分以下: 一般，需谨慎

### Q: 如何切换语言？
A: 点击页面右上角的地球图标，选择 English 或 中文。
